# Nutri-Recommender-Personalized-Food-Recommendation-System-

An AI-powered food recommendation platform that transforms large-scale food and nutrition datasets into personalized meal and item recommendations using hybrid machine learning techniques. The system combines collaborative filtering, content-based modeling, semantic embeddings, and clustering to improve recommendation accuracy and preference matching.

This project integrates recommendation systems, dimensionality reduction, NLP embeddings, similarity modeling, and performance optimization techniques to build a scalable nutrition intelligence engine.
